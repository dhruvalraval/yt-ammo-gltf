# yt-ammo-gltf

Made using three.js and for physics used ammo.js

## Installation

Install locally with npm

```bash
    npm install
```

Start the local server

```bash
    npm run dev
```

Server at: ``` localhost:8080 ```

## Run Dist folder

```bash
    npm install -g http-server
```

```bash
    http-server . -p 8080
```
